# キーボードショートカット

> 作成日: 2026-06-28
> このファイルが**正本**。実装の単一ソースは `frontend/src/shortcuts/registry.ts`。
> アプリ内ヘルプ（`?` キー、または ⌨ ボタン）も同レジストリから生成される。
> 追加・変更時は**本ファイルとレジストリを必ず同時更新**する。
>
> `Mod` = Windows/Linux は **Ctrl**、macOS は **Cmd(⌘)**。
> **文脈依存**: 有効な Window/Dialog によって挙動を切り替える（例: `Esc` はダイアログが開いていれば閉じ、
> ビューアでは表示リセット）。入力欄フォーカス中は `Esc` 以外のショートカットを無効化する。

## グローバル（実装済み）
| キー | 動作 |
|---|---|
| `Mod + ,` | 環境設定を開く |
| `Mod + Shift + D` | DB テーブル管理を開く（standalone のみ） |
| `?` | キーボードショートカット一覧 |
| `Esc` | ダイアログを閉じる |

## ツール（将来・予約：2D ビューア実装時に配線）
| キー | 動作 |
|---|---|
| `W` | Window/Level（階調調整。ドラッグで明るさ/コントラスト） |
| `P` | Pan（移動） |
| `Z` | Zoom（拡大縮小） |
| `L` | Length（2点間の距離計測） |
| `A` | Angle（3点の角度計測） |
| `R` | ROI 矩形（Rectangle） |
| `E` | ROI 楕円（Ellipse） |
| `C` | Crosshairs（MPR 連動十字線） |

## 画像ナビゲーション（将来・予約）
| キー | 動作 |
|---|---|
| `↓` / `↑` | 次 / 前のスライス |
| `→` / `←` | 次 / 前のシリーズ |
| `Space` | シネ再生 / 停止 |
| `Home` / `End` | 最初 / 最後のスライス |

## 表示調整・リセット（将来・予約）
| キー | 動作 |
|---|---|
| `Esc` | 表示のリセット（※ダイアログが開いていれば先に閉じる） |
| `I` | 白黒反転（Invert） |
| `O` | オーバーレイ情報 表示 / 非表示 |
| `F` | フルスクリーン |

## 汎用・システム（将来・予約）
| 機能 | Windows / Linux | macOS |
|---|---|---|
| 元に戻す（Undo） | `Ctrl + Z` | `⌘ + Z` |
| やり直し（Redo） | `Ctrl + Y` | `⌘ + ⇧ + Z` |
| 選択削除（計測/ROI） | `Delete` | `Delete` |
